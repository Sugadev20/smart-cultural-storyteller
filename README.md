# Smart Cultural Storyteller

An automated AI-based storytelling system that generates short cultural stories
and converts them into images, narration, and a final video.

## Features
- Multiple storytelling modes (Mythological, Cultural, Emotional, Ancestral)
- AI-generated story using LLaMA 3 (Groq API)
- Image generation using Stable Diffusion XL
- Text-to-speech narration
- Automatic video creation with captions

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Add your Groq API key
4. Run all cells sequentially

## Output
- Scene-wise images
- Narrated audio
- Final story video with captions
